# Password_Vault
This is a simple GUI based password_vault where you can store your username and password , so that u can retrieve if u forgot...

Libraries : Tkinter
            Hashlib
            sqlite3
            functiontools partial()
            
Database - SQLite
            
This program is splitted into different files just easily debug the code,clone all files to properly installation of program

The main.py file is a root file 
Make sure that all the above mentioned libraries are pre installed
